我哋傾下專案係點運作，同埋點解佢會喺實際製作之上多一層。呢度嘅核心諗法，就係俾你一個層面，可以喺任何 agent 啟動之前、喺花到一蚊之前，先去規劃、反覆修改，同團隊一齊協作。

先講最基本嘅分別。專案係一個容器。建立係免費嘅，會長期保存，亦都係為協作而設。至於製作，就係一次執行。呢部分先至真係要付費，會運行 agent，亦會產出真正嘅成品。

想像有個專案叫做「Brand Campaign Q3」。建立佢免費，保存佢免費，喺入面做規劃都係免費。喺呢個專案入面，你可能會放共用素材，例如品牌套件、聲音同風格參考。你有一個團隊，由一位擁有者加上編輯同審核者組成。仲有一個預算池，譬如預留咗二百四十蚊。亦有一啲預設設定，處理合規、模型同平台之類嘅嘢。然後就到製作本身。其中一個可能係已完成嘅主打影片，花咗六十二蚊。另一個可能係正在運行嘅 TikTok 剪輯版，到目前為止用咗二十八蚊。而第三個可能仲停喺草稿階段嘅培訓內容，暫時一蚊都未使。

呢度有條規則將全部嘢串連埋一齊。喺你明確撳落啟動之前，係唔會使到一蚊。喺嗰一刻之前發生嘅所有嘢，都係完全免費嘅準備工夫。

咁由零去到一個運行中嘅製作，整個流程係點樣呢？首先你建立一個專案，即時完成而且免費。然後你做準備，收集素材、邀請團隊、揀設定，呢個過程可以花幾日甚至幾個禮拜，仍然係零成本。接住你建立一個製作草稿，brief 會儲存低，亦可以繼續修改，同樣免費。之後你攞一個成本估算，由規劃器提供嘅預覽，亦都係免費。然後就到啟動，呢一刻先至開始使錢，agent 亦開始運作。由嗰度開始，你一路監察進度、一路審批，製作就會運行。最後就交付完成。當你準備好做下一個，只需要由草稿嗰步再重複一次。

而家我哋逐個畫面睇下。儀表板就係你嘅起點。最上面有個區叫做「我嘅專案」，加上一個建立新專案嘅按鈕。每個專案都會以卡片形式出現。Brand Q3 可能顯示三個製作，其中兩個運行緊、用咗九十蚊，幾分鐘前更新過。隔籬有個叫 Luna Short Film 嘅專案，顯示一個已完成嘅製作、九十五蚊，三日前更新過。而且永遠都有一張卡俾你開始一個全新嘅專案。

下面你會見到一啲活躍中嘅製作，係由你所有專案集合而成，咁你就可以一眼睇晒所有進行緊嘅嘢，每個都附帶一條小小嘅進度條。再嚟仲有個快速開始嘅區域，係俾你淨係想快快手整啲嘢、唔想做設定嗰陣用。你揀一個範本，系統就會喺背後幫你處理好個專案。啲範本涵蓋鈎子、用戶自製內容、解說片、生日影片、電影等等。

當你真係建立一個新專案嗰陣，會彈出一個對話框。你幫佢改個名。你可以加一段選填嘅描述，譬如話一個橫跨社交媒體同網頁嘅 Q3 品牌認知活動。然後就到預算池，你喺度設定入面所有製作嘅總預算，或者你可以揀無上限，純粹用幾多畀幾多。你亦會揀一個付款方式，例如一張存檔嘅信用卡。

團隊嗰部分係選填，你隨時可以遲啲再加人。你預設係擁有者，你可以用電郵邀請其他人，並分配角色。啲角色值得了解一下。擁有者有完整控制權，包括付款同刪除。編輯可以建立同啟動製作，並管理素材。審核者可以查看、留言同審批關卡。而檢視者就只有唯讀權限。

仲有一個預設設定嘅部分，會套用喺每個製作，除非你逐個覆寫。你可以設定合規選項，例如 C2PA、無障礙標準、演員協議或者私隱法規。你揀一個模型偏好，睇你想要成本優先、品質優先定速度優先。你亦可以而家就連接品牌套件，或者遲啲先加。底部有個友善提示：建立專案係免費嘅，只有當你啟動製作嗰陣先會收費。

一旦你入到個專案，就會去到專案工作區。最上面係專案名稱，附有封存佢或者開啟設定嘅選項。仲有一行分頁：概覽、製作、素材、團隊、設定同活動。

喺概覽分頁，你會一眼睇到一組狀態卡。其中一張顯示你嘅製作，可能有三個，一個完成、一個運行緊、一個喺草稿。另一張顯示預算，二百四十蚊入面用咗九十蚊，大約三成八。第三張顯示團隊，可能有三位成員。第四張顯示你嘅素材，譬如十二個檔案。

卡片下面就係製作清單本身，附有一個開始新製作嘅按鈕，再加上排序同篩選。每個製作都列得好清楚。主打影片標示為完成，係一個 Type E 嘅作品，兩日前完成，花咗六十二蚊，片長五分二十秒，已交付去 YouTube 同 TikTok，仲有選項可以查看成品、睇分析、或者複製成草稿。TikTok 剪輯版運行緊，係一個 Type A 嘅作品，正喺製作階段，用咗二十八蚊，完成咗五成五，大約仲爭兩分鐘，你可以開佢嘅主控台或者暫停佢。培訓單元仲係草稿，係一個 Type F 嘅作品，仲未啟動，零成本，brief 完成咗八成，估算成本大約三十五蚊，提供按鈕去編輯 brief、攞估算、啟動或者刪除。

再下面就係最近活動嘅動態。佢可能會記低幾分鐘前編輯 agent 喺 TikTok 嗰個作品完成咗粗剪，五分鐘前你審批咗佢第一個關卡，兩日前主打影片交付咗去 YouTube 同 TikTok，三日前 Sarah 以審核者身份加入。

素材分頁就係你嘅共用素材庫。最上面有按鈕去上載素材或者連接品牌套件。品牌套件嗰部分放住你嘅標誌、字型、顏色定義、一份指南同一份語調文件，而呢啲全部都會自動載入每個製作嘅品牌 agent。然後仲有個聲音庫。你可能有一把品牌聲音係充滿活力又年輕嘅，透過聲音服務複製出嚟。另一把係權威又成熟嘅，特別訓練出嚟。仲有一把中性、清晰嘅旁白，用標準嘅文字轉語音。你隨時都可以加多啲聲音。

仲有一個風格參考嘅部分，放住譬如一張形容為新黑色電影風格嘅情緒圖、一段節奏參考影片，同一張構圖參考。最後就到劇本同文件嘅區域，放住例如品牌訊息指南、目標受眾研究同競爭對手分析。呢度嘅重點係，呢個素材庫入面嘅所有嘢，都會自動提供俾專案入面每一個製作使用，而啲 agent 引用佢哋嗰陣，係唔需要任何人重新上載。

跟住就到製作草稿畫面，喺度你可以喺啟動之前編輯所有嘢。佢嘅核心係 brief 編輯器。你會見到揀咗嘅範本，今次係企業培訓，你可以更改。你幫個製作改個標題，譬如「培訓單元：產品入門」。然後就到願景，一段文字描述，例如一段十分鐘嘅互動培訓影片，教新員工點用我哋嘅產品儀表板，效果應該要現代、親切，仲要包含知識測驗。

由嗰度你設定啲細節。片長十分鐘、闊銀幕格式、語調親切又專業。你列出學習目標，例如瀏覽儀表板同建立報告。你揀一個評估方式，可能係每節之後一個測驗。你亦揀交付格式，例如常見嘅電子學習打包標準或者獨立影片。你可以由專案拉入參考素材，譬如品牌套件同一把指定嘅品牌聲音，你亦可以加自己嘅額外嘢，例如一個產品截圖嘅壓縮檔。你幫呢個製作設定一個預算，譬如三十五蚊，你亦見到預算池二百四十蚊入面仲剩番一百五十。所有嘢都會自動儲存，所以佢會話幾分鐘前最後儲存過。

然後就到成本估算預覽。你隨時可以重新整理佢，佢亦會顯示上次幾時計算。佢將成本逐項拆開：劇本同教學設計幾蚊、渲染嗰段十分鐘嘅虛擬主播大約十二蚊、聲音合成四蚊、動態圖像六蚊、生成評估兩蚊、品質管控同合規三蚊、打包去學習系統一蚊。加埋總估算大約三十一蚊，上下浮動一成五左右。佢亦話你知需要幾多個 agent，大約十八個，同埋估算嘅時長，大概八分鐘。

仲有一個部分係團隊喺草稿上面嘅留言。審核者 Sarah 可能問你可唔可以加一節講報告匯出，而你回覆話呢個主意好，已經加咗入學習目標。任何人都可以參與發言。

接住就到啲操作。你可以儲存草稿、分享俾人審核，或者啟動製作。啟動按鈕對佢會做啲乜講得好清楚。佢會由專案預算池收取最多三十五蚊。佢會啟用十八個 agent，涵蓋教學設計、虛擬主播、聲音、學習系統等等。佢開始製作，而你會喺製作主控台監察佢。佢估算大約八分鐘完成。

退後一步,睇下整個資訊架構係點砌埋一齊。最頂層係儀表板,佢容納你嘅專案網格、橫跨所有嘢嘅活躍製作,同埋快速開始。然後就到專案工作區,呢個係新嘅,容納概覽、製作清單、共用素材、團隊、設定同活動記錄。仲有製作草稿,亦都係新嘅,容納 brief 編輯器、成本估算、團隊留言同啟動按鈕。啟動之後你就會去到製作主控台,呢個本身已經存在,有佢嘅 agent 圖譜畫布、時間軸檢視、成品畫廊、評論動態、關卡控制同 agent 檢查器。圍住呢一切嘅,就係本身已有嘅區域:agent 登記冊、記憶同知識、交付中心、設定同管理、以及說明同文件。

了解一下啲資料嘅形態都有用,至少喺概念上。一個專案帶住一個身份同名稱、一段選填嘅描述、一位擁有者,同埋一個狀態,唔係活躍就係已封存。佢追蹤佢嘅預算池,呢個可以係空值,即係冇上限,連同已使用咗幾多同付款方式。佢容納佢嘅成員,每位都有一個用戶、一個角色,同加入嘅日期。佢容納共用素材,每件都標示類型,係品牌套件、聲音、風格參考、文件定劇本,附有名稱、位置同一啲中繼資料。佢帶住合規、模型策略同平台目標嘅預設設定。佢亦記錄佢幾時建立同最後更新。

至於一個製作,佢就屬於某個專案。佢有一個標題同一個範本,以 A 到 J 嘅字母標識。佢會經歷一連串狀態:草稿、啟動中、運行中、暫停、已完成或者失敗。佢容納一個 brief,喺仲係草稿嗰陣可以繼續修改,捕捉願景、一個選填嘅類型、片長、長寬比、語調、目標受眾、按素材嘅參考、任何限制、平台目標,同範本特定嘅細節例如學習目標或者節拍數。佢追蹤一個預算上限同已使用嘅金額。佢可以容納由規劃器預覽嚟嘅一個估算,包括低、中、高三個成本數字、需要嘅 agent 數量、一個估算時長、按類別嘅拆解,同佢幾時生成。佢保留團隊喺草稿上嘅留言。而只有喺啟動之後,佢先會多咗例如 agent 圖譜、成品,同啟動同完成嘅時間戳。
