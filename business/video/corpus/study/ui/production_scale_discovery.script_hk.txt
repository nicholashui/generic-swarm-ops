我哋傾下個系統點樣幫人搞清楚自己究竟想整啲乜。喺投放任何預算、啟動任何一個 agent 之前，用家可以先瀏覽真實例子，了解唔同嘅製作規模，再揀返最啱自己諗法嘅一個。一切都係要將探索、靈感同清晰擺喺最前面。

咁我哋要解決嘅問題係乜呢？想像一下而家嘅流程。新用家建立一個專案，緊接住就要喺一張由 A 到 J 嘅清單入面揀一個範本，跟住再設定時長。但佢哋心入面其實諗緊乜呢？佢哋會諗，我根本唔知 Template E 係咩意思。三十秒夠唔夠表達我嘅諗法呀？一個五十蚊嘅製作同二百蚊嘅製作，睇落究竟有咩分別？我可唔可以直接睇下呢個系統實際整到啲咩出嚟？呢個就係空隙所在。用家真正需要嘅，係先瀏覽，攞靈感，了解規模，然後先至開始做設定。

答案就係一個全新嘅頂層分區，叫做 Discovery Hub，即係探索中心。喺側邊導覽列入面，除咗儀表板、專案、製作、agents 同設定之外，而家多咗一個入口，簡單叫做 Discover。佢將藝廊、展示同範本全部集中喺同一個地方。

喺 Discovery Hub 入面，你會搵到一個搜尋列同一組分頁。有 Showcase Gallery 展示藝廊、Templates 範本、Community 社群，仲有一個 Scale Guide 規模指南。展示藝廊就係核心所在，佢係一個精選嘅集合，展示系統實際可以製作出嚟嘅真實例子。你可以按時長、風格、行業同預算範圍嚟篩選，亦都可以按熱門、最新、最多人讚好、最平、或者最高質素嚟排序。

每一個例子都會以卡片形式出現，附帶一段可以播放嘅預覽。睇下其中幾個。有一個叫 Neon Dreams，係一段十五秒嘅 hook，花咗大約八蚊，用 Template A 整出嚟，攞到五星。隔籬一個叫 Product Launch，係三十秒嘅用家自製風格作品，大約十五蚊，用 Template B。再嚟一個叫 Five Minute Doc 嘅五分鐘紀錄片，大約四十五蚊，用 Template I。仲有一條叫 Brand Story 嘅九十秒品牌片，大約三十五蚊，用 Template E。下面你仲會見到更多，例如一段二十秒嘅健身廣告，大約十二蚊；一段三分鐘嘅 AI 虛擬主播對談，大約二十五蚊；一條四分鐘嘅音樂影片，接近六十蚊；同埋一段十二分鐘嘅企業培訓片，大約四十蚊。任何時候你可能只係見到二百四十七個展示項目入面嘅其中八個，但你可以揀載入更多。撳任何一張卡片，你就會見到完整預覽，仲有一個掣寫住「以此作為起點」。

當你真係撳入一個展示，你就會見到完整嘅詳情頁面。我哋繼續用 Neon Dreams 做例子。最頂有一個影片播放器，播緊嗰段十五秒嘅片。下面就係製作詳情。佢係 Template A，即係 Viral Hook 病毒式爆紅 hook。時長十五秒。畫面比例係九比十六，專為 TikTok 同 Reels 而做。風格係賽博龐克、霓虹、高能量。類型係生活時尚。

跟住有一個規模拆解，呢個真係好實用。呢個製作用咗十二個 agent，生成咗四個鏡頭。總成本係八蚊二毫。製作時間係兩分十四秒。主要模型係 Veo 三點一，由 Kling 三點零負責拍 B roll 補充鏡頭。冇旁白，全程只有音樂，而音樂本身就由 Udio 生成，係一段每分鐘一百三十拍嘅電子曲。

你亦都會見到實際參與嘅每一個 agent。有 Director Agent 導演、Prompt Engineer Agent 提示工程師、Editor Agent 剪接、Composer Agent 作曲，同埋 AI QA Agent 品質檢查。再加埋 Retention Optimizer Agent 留存優化、Trend Intel Agent 趨勢情報、Copywriter Agent 文案，同 Social Strategist Agent 社交策略。最後仲有 Colorist Agent 調色、Sound Mixer Agent 混音，同埋 Motion Graphics 動態圖像 agent。

詳情頁面甚至會顯示當初用嘅製作簡介。佢係咁寫嘅，為一個時裝品牌而設嘅十五秒 hook，夜晚嘅霓虹城市街道，一個模特兒行緊路，配合節拍剪接，採用熱門音訊風格，目標受眾係 Z 世代。

然後就係你可以採取嘅行動。你可以用呢個做範本，咁就會建立一個已經填好嗰份簡介嘅草稿。你可以將簡介文字複製到剪貼簿。你可以將佢 remix 改編，意思即係由呢個出發，但改一改風格、時長或者受眾。又或者你可以一撳就將佢儲存到你嘅靈感板。

而家睇下 Scale Guide 規模指南呢個分頁，佢就係回答「我應該整啲乜」呢條問題。佢列出咗完整嘅製作規模光譜，由五秒一直去到兩個鐘。沿住呢條線排住一啲有名嘅級別。有 Micro 微型，跟住係 Hook，再嚟係 Short 短片，然後係 Medium 中型，跟住係 Long 長片，再有 Episodic 連續劇集，最後就係 Feature 長篇。每一個級別都附帶一個大概嘅成本同所需投入嘅感覺。Micro 大約兩至五蚊，用大概四個 agent。Hook 落喺五至十五蚊，用八至十二個 agent。Short 係十至三十蚊，用十二至十八個 agent。Medium 係二十至五十蚊，用十八至三十個 agent。Long 係四十至八十蚊，用三十至五十個 agent。Episodic 由大約六十蚊起，用五十至八十個 agent。而 Feature 就由一百五十蚊起跳，要動用一百一十四個 agent。撳任何一個級別，你就會見到喺嗰個規模下製作出嚟嘅例子。

仲有一個詳細比較，逐個規模並排咁講解。一個 Micro Clip 微型短片係三至五秒，兩至五蚊，一分鐘以內完成，用四至六個 agent，最啱攞嚟取代 GIF 或者整循環動畫。一個 Viral Hook 係七至十五秒，五至十五蚊，一至三分鐘，八至十二個 agent，係 TikTok、Reels 同 YouTube Shorts 嘅首選，用 Template A。一個 Short Ad 短廣告係十五至四十五秒，十至三十蚊，兩至五分鐘，十二至十八個 agent，最適合用家自製風格嘅廣告、產品示範同見證式內容，用 Template B。一個 Explainer 講解片係六十至一百八十秒，二十至五十蚊，三至八分鐘，十八至二十五個 agent，最啱做產品講解同動畫教學，用 Template C。

繼續落去，一條 Music Video 音樂影片係三至五分鐘，四十至八十蚊，製作要八至十五分鐘，二十五至三十五個 agent，最適合完整音樂影片、視覺專輯或者歌詞片，用 Template G。一條 Short Film 短片係三至十五分鐘，五十至九十五蚊，製作要十至二十五分鐘，三十至五十個 agent，啱做敘事作品、品牌片同短紀錄片，用 Template E。一段 Training 培訓片係五至三十分鐘，三十至六十蚊，八至二十分鐘，十八至三十個 agent，為企業內容、學習管理系統同入職培訓而設，用 Template F。一齣 Documentary 紀錄片橫跨十至九十分鐘，八十蚊起，製作要三十分鐘或以上，四十至六十個 agent，做訪談式、檔案式同調查式作品，用 Template I。而喺最頂端，一齣 Feature 長篇就係九十分鐘或以上，一百五十蚊起，製作超過一個鐘，要一百一十四個 agent，做完整敘事電影同劇集，用 Template J。

接住就係 Smart Selection Wizard 智能選擇精靈，係一個由 AI 引導嘅方式嚟回答同一條問題，我應該整啲乜。喺呢度你只需要用普通說話描述你嘅諗法，系統就會推薦返啱嘅規模。想像有人打咗，我想要一段宣傳片，介紹我間咖啡店嘅新季度飲品，感覺要溫馨而又貼潮流，畀 Instagram 同 TikTok 用。佢撳一撳建議，AI 就會回應。

根據呢段描述，最頂嘅推薦係用 Template A 嘅 Viral Hook。建議時長係十五至三十秒，直度九比十六，以社交平台為先。預估成本大約十二蚊，製作時間大約三分鐘。佢仲會解釋點解，因為附帶熱門 hook 嘅短片社交內容，喺 Instagram 同 TikTok 上面可以攞到最大嘅觸及。佢甚至會提供一個相似展示嘅預覽，同埋一個可以即刻採用嘅掣。

呢個精靈亦都會提出值得考慮嘅其他選擇。其中一個係用 Template B 嘅用家自製風格廣告，三十秒，大約十八蚊，感覺更加真實，帶見證式風格，如果你想要嗰種「真人講嘢」嘅感覺就好啱。另一個係一個多格式組合，將 hook、廣告同故事捆綁埋一齊，三條影片分別係十五秒、三十秒同六十秒版本，全部出自同一份簡介，總共大約三十五蚊，涵蓋所有投放位置。

跟住就係 Community 社群分頁，係一個由其他用家製作嘅展示動態消息。你可以按熱門、最新、本週最佳，或者編輯精選嚟排序，亦可以按類型、時長同風格嚟篩選。佢以類似 Pinterest 嘅瀑布流網格排版，影片有各種形狀，闊嘅十六比九、長嘅九比十六，同埋正方形嘅一比一。你會見到一啲作品，例如由用家 Luca 整嘅四十五秒短片 Midnight Cafe，攞到三百四十二個讚好；由 Maya 整嘅十五秒 hook Dance Challenge，攞到一千二百個讚好同八十九個留言；仲有一個由品牌帳號整嘅二十秒 Product Launch，攞到一百二十八個讚好。仲有一段由企業帳號整嘅八分鐘安全培訓片、一段由 Sam 整嘅兩分鐘 AI 虛擬主播，同埋一條由 Beats 整嘅三分鐘音樂影片改編，攞到超過五百個讚好。撳任何一個，你都可以預覽、改編，或者沿用相同設定。

Templates 範本分頁就以一個可以左右捲動嘅視覺輪播形式，呈現各種製作類型。每一個範本都有自己嘅卡片同預覽。Template A 係 Viral Hook，七至六十秒，直度，五至十五蚊，大約兩分鐘整完，最啱 TikTok、Reels 同 YouTube Shorts。Template B 係用家自製風格嘅 Ad 廣告，十五至四十五秒，直度，十至三十蚊，大約四分鐘，為付費社交同 Meta、TikTok 上面嘅廣告而設。Template C 係 Animated Explainer 動畫講解片，六十至一百八十秒，闊銀幕十六比九，二十至五十蚊，大約七分鐘，最適合網站、YouTube 同 SaaS 示範。每張卡片都畀你採用嗰個範本，或者睇返幾個例子。而個輪播會繼續落去，有 Template D 做生日、E 做電影、F 做培訓、G 做音樂影片、H 做虛擬主播、I 做紀錄片，同 J 做長篇電影。

咁個系統實際上係點樣搵出啱嘅規模呢？當用家描述佢嘅諗法，或者揀咗一個展示，系統就會行一個快速又平嘅推斷步驟。可以當佢係一個影片製作顧問。畀咗一段關於某人想創作啲乜嘅描述，佢就會推薦最好嘅製作規模同範本。佢會權衡平台，因為社交即係短片，網頁即係中型，戲院即係長片。佢會考慮內容類型，廣告係十五至六十秒，講解片係一至三分鐘，敘事就係三分鐘或以上。佢會計埋複雜度，講話人頭係簡單，視覺特效係複雜，多場景就代表更長嘅作品。佢仲會顧及受眾期望同對預算嘅敏感程度。由所有呢啲嘢出發，佢會回傳一個推薦範本、一個時長範圍、一個低中高三檔嘅成本估算、一個預估製作時間、所需 agent 數目、一個畫面比例、佢嘅推理，同埋一組其他選擇，每一個都附帶自己嘅理據同取捨。

仲有一個 Inspiration Board 靈感板，佢存在於每一個專案入面。喺整任何嘢之前，用家可以儲存展示項目作參考。想像一個叫 Coffee Shop Campaign 嘅專案。佢嘅靈感板入面放住啲由 Discover 儲存返嚟嘅項目，例如儲低 Neon Dreams 係為咗佢嘅風格、一段叫 Cozy Vibes 嘅三十秒片係為咗佢嘅基調、一段二十秒嘅 Latte Art 片係為咗佢嘅剪接，旁邊仲有一個掣可以由 Discover 瀏覽更多。用家甚至留低咗一個畀自己嘅筆記。佢寫住，我鍾意 Cozy Vibes 嗰種暖色調，但又想要 Neon Dreams 嗰種快節奏，而 Latte Art 就有啱嘅、聚焦產品嘅取景。由嗰度，一個動作就可以由呢啲參考建立一個製作，而系統會自動將從儲存項目抽出嚟嘅風格線索填入簡介。

我哋退後一步，睇下呢個點樣改變成個用家旅程。以前係好令人困惑嘅。你建立一個專案，被叫去喺 A 到 J 入面揀一個範本，揀到一頭霧水，最後往往就索性放棄。之後就變成有引導嘅。你建立一個專案，瀏覽 Discovery Hub，睇到真實例子，然後諗，就係嗰個，我想要好似嗰個咁嘅嘢。你撳用作範本，份簡介就會以啱嘅規模預先填好，你調整下細節，然後就推出。

又或者你行 AI 引導嗰條路。你建立一個專案，用普通英文描述你嘅諗法，AI 就推薦，你需要一段十五秒嘅 hook，而原因係咁。佢會喺嗰個規模下畀三個例子你睇，你揀一個，份簡介就預先填好，然後推出。又或者你行社群啟發嗰條路。你瀏覽社群藝廊，睇到熱門內容，決定想整啲好似咁嘅嘢，撳改編，分支佢嘅簡介，調整一下，然後推出。

最後總結一下，系統可以用嚟搵出你製作規模嘅幾種方式。Template Carousel 範本輪播畀你瀏覽展示時長、成本同例子嘅視覺卡片，付出嘅努力好低。Scale Guide 規模指南畀你一個將每個製作級別並排嘅比較，同樣低努力，睇完再揀就得。Showcase Gallery 展示藝廊畀你睇真實製作出嚟、附完整中繼資料嘅例子，除咗睇之外幾乎唔使花任何功夫。Smart Wizard 智能精靈畀你描述諗法，由 AI 推薦規模，努力極微，打一句嘢就得。Community Feed 社群動態畀你瀏覽其他人整過嘅嘢，再分支或者改編，同樣係零功夫。Inspiration Board 靈感板畀你隨住時間儲存參考，而系統會從呢個集合推斷你嘅風格同規模，係好自然咁發生。而由一個展示出發，撳用呢個做範本，就會一撳填入確切嘅設定。

呢啲路徑每一條都通往同一個終點。一個草稿製作，已經填好正確嘅範本、時長、畫面比例同預算，準備好畀用家自訂同推出。
