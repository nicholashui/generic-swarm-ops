我们来聊聊这个系统是怎么帮人搞清楚自己究竟想做什么的。在投入任何预算、启动任何一个 agent 之前，用户可以先浏览真实的例子，了解不同的制作规模，再挑出最符合自己想法的那一个。这一切的核心，就是把探索、灵感和清晰放在最前面。

那么我们要解决的问题是什么呢？想象一下现在的流程。新用户创建一个项目，紧接着就要在一份从 A 到 J 的清单里选一个模板，然后再设置时长。但他们心里其实在想什么呢？他们会想，我根本不知道 Template E 是什么意思。三十秒够不够表达我的想法？一个五十美元的制作和两百美元的制作，看起来到底有什么区别？我能不能直接看看这个系统实际能做出什么来？这就是缺口所在。用户真正需要的，是先浏览，找灵感，了解规模，然后再开始做设置。

答案就是一个全新的顶层板块，叫做 Discovery Hub，也就是探索中心。在侧边导航栏里，除了仪表盘、项目、制作、agents 和设置之外，现在多了一个入口，简单叫做 Discover。它把画廊、展示和模板全部集中在同一个地方。

在 Discovery Hub 里，你会看到一个搜索栏和一组标签页。有 Showcase Gallery 展示画廊、Templates 模板、Community 社区，还有一个 Scale Guide 规模指南。展示画廊就是核心所在，它是一个精选的合集，展示系统实际能制作出来的真实例子。你可以按时长、风格、行业和预算范围来筛选，也可以按热门、最新、最多人点赞、最便宜，或者最高质量来排序。

每一个例子都会以卡片的形式出现，并附带一段可以播放的预览。看看其中几个。有一个叫 Neon Dreams，是一段十五秒的 hook，花了大约八美元，用 Template A 做出来，拿到五星。旁边一个叫 Product Launch，是三十秒的用户自制风格作品，大约十五美元，用 Template B。再来一个叫 Five Minute Doc 的五分钟纪录片，大约四十五美元，用 Template I。还有一条叫 Brand Story 的九十秒品牌片，大约三十五美元，用 Template E。下面你还会看到更多，比如一段二十秒的健身广告，大约十二美元；一段三分钟的 AI 虚拟主播对谈，大约二十五美元；一条四分钟的音乐视频，接近六十美元；以及一段十二分钟的企业培训片，大约四十美元。任何时候你可能只看到两百四十七个展示项目里的其中八个，但你可以选择加载更多。点击任何一张卡片，你就会看到完整预览，还有一个按钮写着「以此作为起点」。

当你真的点进一个展示，你就会看到完整的详情页面。我们继续用 Neon Dreams 做例子。最上面有一个视频播放器，正在播放那段十五秒的片子。下面就是制作详情。它是 Template A，也就是 Viral Hook 病毒式爆款 hook。时长十五秒。画面比例是九比十六，专为 TikTok 和 Reels 而做。风格是赛博朋克、霓虹、高能量。类型是生活时尚。

接着有一个规模拆解，这个真的很实用。这个制作用了十二个 agent，生成了四个镜头。总成本是八美元两毛。制作时间是两分十四秒。主要模型是 Veo 三点一，由 Kling 三点零负责拍 B-roll 补充镜头。没有旁白，全程只有音乐，而音乐本身由 Udio 生成，是一段每分钟一百三十拍的电子曲。

你还会看到实际参与的每一个 agent。有 Director Agent 导演、Prompt Engineer Agent 提示工程师、Editor Agent 剪辑、Composer Agent 作曲，以及 AI QA Agent 质量检查。再加上 Retention Optimizer Agent 留存优化、Trend Intel Agent 趋势情报、Copywriter Agent 文案，和 Social Strategist Agent 社交策略。最后还有 Colorist Agent 调色、Sound Mixer Agent 混音，以及 Motion Graphics 动态图形 agent。

详情页面甚至会显示当初用的制作简报。它是这么写的，为一个时尚品牌而做的十五秒 hook，夜晚的霓虹城市街道，一个模特在走路，配合节拍剪辑，采用热门的音频风格，目标受众是 Z 世代。

然后就是你可以采取的操作。你可以把这个用作模板，这样就会创建一个已经填好那份简报的草稿。你可以把简报文字复制到剪贴板。你可以对它进行 remix 改编，意思就是从这个出发，但改一改风格、时长或者受众。或者你也可以一点就把它保存到你的灵感板。

现在看看 Scale Guide 规模指南这个标签页，它就是回答「我该做什么」这个问题的。它列出了完整的制作规模谱系，从五秒一直到两个小时。沿着这条线排着一些有名字的级别。有 Micro 微型，接着是 Hook，再来是 Short 短片，然后是 Medium 中型，接着是 Long 长片，再有 Episodic 连续剧集，最后就是 Feature 长篇。每一个级别都附带一个大概的成本和所需投入的感觉。Micro 大约两到五美元，用大约四个 agent。Hook 落在五到十五美元，用八到十二个 agent。Short 是十到三十美元，用十二到十八个 agent。Medium 是二十到五十美元，用十八到三十个 agent。Long 是四十到八十美元，用三十到五十个 agent。Episodic 从大约六十美元起，用五十到八十个 agent。而 Feature 则从一百五十美元起步，要动用一百一十四个 agent。点击任何一个级别，你就会看到在那个规模下制作出来的例子。

还有一个详细比较，把每个规模并排讲解。一个 Micro Clip 微型短片是三到五秒，两到五美元，一分钟以内完成，用四到六个 agent，最适合用来替代 GIF 或者做循环动画。一个 Viral Hook 是七到十五秒，五到十五美元，一到三分钟，八到十二个 agent，是 TikTok、Reels 和 YouTube Shorts 的首选，用 Template A。一个 Short Ad 短广告是十五到四十五秒，十到三十美元，两到五分钟，十二到十八个 agent，最适合用户自制风格的广告、产品演示和见证式内容，用 Template B。一个 Explainer 讲解片是六十到一百八十秒，二十到五十美元，三到八分钟，十八到二十五个 agent，最适合做产品讲解和动画教程，用 Template C。

继续往下，一条 Music Video 音乐视频是三到五分钟，四十到八十美元，制作要八到十五分钟，二十五到三十五个 agent，最适合完整音乐视频、视觉专辑或者歌词片，用 Template G。一条 Short Film 短片是三到十五分钟，五十到九十五美元，制作要十到二十五分钟，三十到五十个 agent，适合叙事作品、品牌片和短纪录片，用 Template E。一段 Training 培训片是五到三十分钟，三十到六十美元，八到二十分钟，十八到三十个 agent，为企业内容、学习管理系统和入职培训而做，用 Template F。一部 Documentary 纪录片横跨十到九十分钟，八十美元起，制作要三十分钟或以上，四十到六十个 agent，做访谈式、档案式和调查式作品，用 Template I。而在最顶端，一部 Feature 长篇就是九十分钟或以上，一百五十美元起，制作超过一个小时，要一百一十四个 agent，做完整叙事电影和剧集，用 Template J。

接着就是 Smart Selection Wizard 智能选择向导，它是一种由 AI 引导的方式，来回答同一个问题，我该做什么。在这里你只需要用平常的话描述你的想法，系统就会推荐合适的规模。想象有人输入了，我想要一段宣传片，介绍我咖啡店的新季节饮品，感觉要温馨又时尚，给 Instagram 和 TikTok 用。他点一下建议，AI 就会回应。

根据这段描述，最顶端的推荐是用 Template A 的 Viral Hook。建议时长是十五到三十秒，竖版九比十六，以社交平台为先。预估成本大约十二美元，制作时间大约三分钟。它还会解释原因，因为带有热门 hook 的短视频社交内容，在 Instagram 和 TikTok 上能拿到最大的触达。它甚至会提供一个相似展示的预览，以及一个可以马上采用的按钮。

这个向导还会提出值得考虑的其他选项。其中一个是用 Template B 的用户自制风格广告，三十秒，大约十八美元，感觉更真实，带见证式风格，如果你想要那种「真人说话」的感觉就很合适。另一个是一个多格式组合，把 hook、广告和故事打包在一起，三条视频分别是十五秒、三十秒和六十秒版本，全部出自同一份简报，总共大约三十五美元，覆盖所有投放位置。

接下来就是 Community 社区标签页，它是一个由其他用户制作的展示信息流。你可以按热门、最新、本周最佳，或者编辑精选来排序，也可以按类型、时长和风格来筛选。它以类似 Pinterest 的瀑布流网格排版，视频有各种形状，宽的十六比九、长的九比十六，以及正方形的一比一。你会看到一些作品，比如由用户 Luca 做的四十五秒短片 Midnight Cafe，拿到三百四十二个赞；由 Maya 做的十五秒 hook Dance Challenge，拿到一千两百个赞和八十九条评论；还有一个由品牌账号做的二十秒 Product Launch，拿到一百二十八个赞。还有一段由企业账号做的八分钟安全培训片、一段由 Sam 做的两分钟 AI 虚拟主播，以及一条由 Beats 做的三分钟音乐视频改编，拿到超过五百个赞。点击任何一个，你都可以预览、改编，或者沿用相同的设置。

Templates 模板标签页则以一个可以左右滑动的视觉轮播形式，呈现各种制作类型。每一个模板都有自己的卡片和预览。Template A 是 Viral Hook，七到六十秒，竖版，五到十五美元，大约两分钟做完，最适合 TikTok、Reels 和 YouTube Shorts。Template B 是用户自制风格的 Ad 广告，十五到四十五秒，竖版，十到三十美元，大约四分钟，为付费社交以及 Meta、TikTok 上的广告而做。Template C 是 Animated Explainer 动画讲解片，六十到一百八十秒，宽屏十六比九，二十到五十美元，大约七分钟，最适合网站、YouTube 和 SaaS 演示。每张卡片都让你采用那个模板，或者查看几个例子。而这个轮播会继续下去，有 Template D 做生日、E 做电影、F 做培训、G 做音乐视频、H 做虚拟主播、I 做纪录片，和 J 做长篇电影。

那么这个系统实际上是怎么找出合适规模的呢？当用户描述他的想法，或者选了一个展示，系统就会跑一个又快又便宜的推断步骤。可以把它当成一个视频制作顾问。给它一段关于某人想创作什么的描述，它就会推荐最好的制作规模和模板。它会权衡平台，因为社交就是短片，网页就是中型，影院就是长片。它会考虑内容类型，广告是十五到六十秒，讲解片是一到三分钟，叙事就是三分钟或以上。它会算上复杂度，讲话人头是简单，视觉特效是复杂，多场景就意味着更长的作品。它还会顾及受众期望和对预算的敏感程度。从所有这些出发，它会返回一个推荐模板、一个时长范围、一个低中高三档的成本估算、一个预估制作时间、所需 agent 数量、一个画面比例、它的推理，以及一组其他选项，每一个都附带自己的理由和取舍。

还有一个 Inspiration Board 灵感板，它存在于每一个项目里。在做任何东西之前，用户可以保存展示项目作为参考。想象一个叫 Coffee Shop Campaign 的项目。它的灵感板里放着一些从 Discover 保存回来的项目，比如存下 Neon Dreams 是为了它的风格、一段叫 Cozy Vibes 的三十秒片是为了它的基调、一段二十秒的 Latte Art 片是为了它的剪辑，旁边还有一个按钮可以从 Discover 浏览更多。用户甚至给自己留了一条笔记。它写着，我喜欢 Cozy Vibes 那种暖色调，但又想要 Neon Dreams 那种快节奏，而 Latte Art 有合适的、聚焦产品的取景。从那里，一个操作就可以从这些参考创建一个制作，而系统会自动把从保存项目里抽出来的风格线索填进简报。

我们退后一步，看看这是怎么改变整个用户旅程的。以前是很让人困惑的。你创建一个项目，被叫去在 A 到 J 里选一个模板，选得一头雾水，最后往往就干脆放弃。之后就变成有引导的。你创建一个项目，浏览 Discovery Hub，看到真实的例子，然后想，就是那个，我想要像那个一样的东西。你点用作模板，那份简报就会以合适的规模预先填好，你调整一下细节，然后就发布。

或者你走 AI 引导那条路。你创建一个项目，用平常的英文描述你的想法，AI 就推荐，你需要一段十五秒的 hook，原因是这样。它会在那个规模下给你三个例子看，你选一个，简报就预先填好，然后发布。或者你走社区启发那条路。你浏览社区画廊，看到热门内容，决定想做点像这样的东西，点改编，分叉它的简报，调整一下，然后发布。

最后总结一下，系统可以用来找出你制作规模的几种方式。Template Carousel 模板轮播让你浏览展示时长、成本和例子的视觉卡片，付出的努力很低。Scale Guide 规模指南给你一个把每个制作级别并排的比较，同样省力，看完再选就行。Showcase Gallery 展示画廊让你看真实制作出来、附完整元数据的例子，除了看之外几乎不用花任何功夫。Smart Wizard 智能向导让你描述想法，由 AI 推荐规模，努力极小，打一句话就行。Community Feed 社区信息流让你浏览别人做过的东西，再分叉或者改编，同样是零功夫。Inspiration Board 灵感板让你随着时间保存参考，而系统会从这个合集推断你的风格和规模，是很自然地发生的。而从一个展示出发，点用这个做模板，就会一点填入确切的设置。

这些路径每一条都通向同一个终点。一个草稿制作，已经填好正确的模板、时长、画面比例和预算，准备好让用户自定义和发布。
