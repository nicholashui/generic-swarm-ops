The root is a thin orchestrator: `package.json` exposes npm scripts that invoke Node/Python harnesses under `scripts/` to bootstrap, sync, validate, and audit the system. The harnesses generate two agent-harness trees — `.trae/` for Trae IDE and `.grok/` for Grok Build — from shared source-of-truth directories (`rules/`, `skills/`, `hooks/`, `mcp-configs/`, `scripts/adapters/shared.mjs`). At runtime the Next.js frontend calls the FastAPI server's OpenAPI contract; both consume the same business artifacts under `business/` (domain packs, agent manifests, evaluation suites) and are steered by `governance_rules/`. `examples_and_samples/` drives the starter bootstrap that clones upstream ecosystems into `external/sources/`, which `tests_and_reviews/` validates against the governance and evolution contracts defined in `planning_docs/`. Documentation under `documentation/` and top-level `README.md` / `structure.md` form the single source of truth consumed by agents across both harnesses.
