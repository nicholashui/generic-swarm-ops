`npm run bootstrap` runs the project-starter harness to clone upstream sources into `external/sources/`, then seeds `sources/source-lock.json`, `docs/source-audit.md`, `.trae/`, `.grok/`, `business/`, and `backend/`; `npm run sync` regenerates both harness trees from shared rules/skills/hooks; `npm run business:*` commands drive validation, governance checks, security controls, evolution sandbox checks, and the eval harness.
