# Frontend Project Rules

- Use TypeScript with Next.js App Router.
- Use Tailwind CSS and shared design tokens.
- Use typed backend API clients.
- Do not expose secrets in browser code.
- Mask API keys after initial reveal.
- Add loading, empty, and error states for major pages.
- Because OpenDesign MCP is unavailable in this workspace, use the documented fallback design files until the official server is configured.
